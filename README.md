# ubuntu-dev-env-initializer
Script that installs and sets up everything you need to start developing on your brand new ubuntu installation

> ## Instructions

* 1 - Clone the repo.
* 2 - Run the script: `chmod +x initialize.sh && ./initialize.sh`
